1.操作方法
 移動→WASD
 攻撃→右クリック
 スキル→各QERC
 インベントリ→Tab
 インタラクト→F
 強制シーン遷移（デバッグ用）→F12
 強制会話イベント(デバッグ用)→右Alt
 ゲーム終了→Esc
2.PRポイント
・ストーリー分岐ができます。
　どう分岐するかはみんなの遊び方次第！
・インタラクトできます。
　レイヤーでインタラクトに選択、インターフェースをインタラクトさせたいものに実装するとプレイヤー側の関数は一つで済むように実装しました！
・敵を倒してスキルを奪おう！
　このゲームは敵を倒すことでその敵の持つスキルを奪うことができます。
 　いっぱい倒してスキルを組み合わせ、ゲームを攻略していこう！！
・プレイヤー、エネミーともに自作のステートマシンで動かしてます。
・RPGにしました。
　企画当初ただのアクションゲームにしようとしていましたが、あまりにもつまらないゲームになりそうだったので先生のアドバイスもありストーリーを作ることになりました。

5.提出バージョン
ゲームループはできます。
タイトル→チュートリアル→メインゲーム→エンディング　この順番でゲームがループします。
現在スキルの種類は3種類
会話パート、探索・戦闘パートと別れています。
6.既知の不具合
インベントリが開けません
それに応じてスキルも打てません
一部の敵が固すぎて倒せません
