1.操作方法
 移動→WASD
 攻撃→Q,E,R,C
 インタラクト→F
 シーン遷移→F12
 タイトルシーン後のシーンで会話パートを進める→右クリック
 また会話パート１から会話パート２に移行→H
 ゲーム終了→Esc

 いっぱいスキル使えるようになる（予定）
 エンディング分岐できる（予定）
